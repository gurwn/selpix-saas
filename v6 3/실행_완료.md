## 📋 전체 파일 실행 방법 정리

### 1️⃣ **Node.js 통합 시스템** (v6 폴더) ⭐ **메인 시스템**

#### 실행 방법
```bash
cd "프로토타입/v6/v6"
npm install          # 의존성 설치 (이미 완료)
npm start            # 서버 실행 (현재 실행 중)
```

#### 접근 주소
- **메인 대시보드**: http://localhost:3000
- **API Health Check**: http://localhost:3000/api/v1/system/health
- **API 문서**: http://localhost:3000/api/v1/

#### 주요 기능
- 🔍 **상품 분석**: 도매꾹 URL 입력 → 자동 상품 정보 추출
- 💰 **마진 계산**: 환율, 수수료, 배송비 포함 정확한 마진 계산
- 🤖 **AI 키워드 생성**: GPT 기반 20개 키워드 자동 생성
- 📈 **자동 등록**: 쿠팡 윙 API를 통한 원클릭 상품 등록
- 🤖 **Discord Bot**: 팀 협업을 위한 Discord 통합

#### API 엔드포인트
```bash
POST /api/v1/products/analyze      # 상품 분석
POST /api/v1/margin/calculate      # 마진 계산
POST /api/v1/keywords/generate     # 키워드 생성
POST /api/v1/registration/register # 상품 등록
GET  /api/v1/system/health         # 헬스 체크
```

#### Discord 봇 명령어
```bash
/소싱요청 url:도매꾹URL 플랫폼:wing 목표마진율:30
/소싱조회 요청id:optional
/소싱취소 요청id:required
/소싱템플릿
/도움말
```

## 🔧 서버 관리 명령어

### 서버 중지
현재 백그라운드로 실행 중인 서버를 중지하려면:
```bash
# 프로세스 찾기
tasklist | findstr node

# 프로세스 종료
taskkill /F /IM node.exe
```

### 서버 재시작
```bash
cd "프로토타입/v6/v6"
npm start
```

### 로그 확인
서버 콘솔 출력을 확인하려면 포그라운드로 실행:
```bash
cd "프로토타입/v6/v6"
node server.js
```

## 🐛 문제 해결

### 포트 충돌 (이미 사용 중)
```bash
# 포트 3000을 사용하는 프로세스 찾기
netstat -ano | findstr :3000

# 해당 PID 종료
taskkill /F /PID [PID번호]

# 또는 .env 파일에서 포트 변경
PORT=3001
```

### 의존성 오류
```bash
cd "프로토타입/v6/v6"
npm install --force
```

### 환경 변수 설정
```bash
# .env 파일 편집
notepad .env

# 필수 항목 (선택적):
```
